#  API RESTful - Gestão de Clientes

Uma API REST desenvolvida em Node.js e Express para o gerenciamento (CRUD) de clientes. Este projeto foi construído focando nas melhores práticas de desenvolvimento de APIs modernas.

##  Tecnologias Utilizadas

- **JavaScript / Node.js**
- **Express.js** (Versão 5.2.1)
- **JSON** (Para simulação de banco de dados em memória)

##  Como executar o projeto

1. Clone este repositório.
2. Abra o terminal integrado do seu editor de código.
3. Instale as dependências executando:
   ```bash
   npm install
